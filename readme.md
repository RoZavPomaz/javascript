1.Programación:
Programar es crear un conjunto de instrucciones para que una computadora realice una tarea específica. Implica decirle a la máquina qué hacer y cómo hacerlo.

2.Variables:
Las variables son espacios de almacenamiento que tienen nombres y pueden guardar información. Son como cajas donde puedes poner valores, y esos valores pueden cambiar mientras el programa se ejecuta.

3.Tipos de Datos:
Los tipos de datos son categorías que definen qué tipo de información puede ser almacenada. Pueden ser números, palabras, listas de cosas, etc. Ayudan a la computadora a entender y manejar diferentes tipos de información.

4.Operadores:
Los operadores son herramientas que realizan acciones en la programación. Pueden sumar números, comparar cosas, o asignar valores a variables. Son como las herramientas que usas para construir o cambiar cosas en tu programa.
